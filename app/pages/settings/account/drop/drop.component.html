티샷 회원은 서비스 이용 원하지 않은 경우
언제든지 티샷 가입에 사용된 Apple ID 를 삭제하고 이용 계약 해지를 할 수 있습니다.
Apple ID 를 삭제하면 회원의 서비스이용정보는 모두 삭제되고 복구가 불가능하게 됩니다.

Apple ID를 삭제하더라도 자동 갱신 구독은 사용자가 취소 신청을 하기 전까지는 계속됩니다.
Apple ID 삭제 후 본인이 직접 다음 경로를 통해서 구독을 취소하거나 환불을 요청하셔야 합니다.

* iPhone : 설정 앱 > 본인 이름 탭 > 구독 탭

이용권이 남아 있는 경우,
Apple ID 삭제 시 이용권은 소멸됩니다.

티샷 회원은 서비스 이용을 원하지 않은 경우
언제든지 회원 탈퇴를 통해 이용 계약 해지를 할 수 있습니다.
회원 탈퇴를 하면 회원의 서비스 이용정보는 삭제되어 복구가 불가능하게 됩니다.

정기 구독을 이용하시는 경우 탈퇴 전에 정기구독취소 신청을 해주시고 탈퇴 후 에는 본인이 직접 다음 경로를 통해서 취소하셔야 합니다.

* iPhone : 설정 앱 > 본인 이름 탭 > 구독 탭
* Android 폰: >Play 스토어 > 우측 상단 본인 이름 탭 > 결제 및 정기결제 탭 > 정기결제 탭

이용권이 남아 있는 경우,
회원 탈퇴 시 이용권은 소멸됩니다.
